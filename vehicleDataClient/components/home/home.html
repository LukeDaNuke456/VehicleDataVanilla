Welcome to Vehicle Data Vanilla!

This web application demonstrates that a full-stack application can be built using a Vite Vanilla JavaScript frontend and a Node.js Express backend. It follows an MVC (Model-View-Controller) architecture, with car data stored in a PostgreSQL database, showcasing the effective integration of these technologies.
